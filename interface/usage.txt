Usage: 
    1) python3 interface.py
                               Solve all queries; produce a tex file including
                               preamble and labeled queries; compile the tex 
                               file and open the pdf
    2) python3 interface.py [options]
                               Solve queries and produce/open files according
                               to the options
    3) python3 interface.py --help
                               Display this message
    4) python3 interface.py --version
                               Display the system's version

Options:
    --tex       Only produce a tex file without preamble and labeled queries,
                don't compile or open pdf
    --all       Solve all queries in queries.pl
    -q i        Solve only query number i
    -q i j      Solve queries i until and including j
